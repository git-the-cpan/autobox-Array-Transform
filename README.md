p5-autobox-Array-Transform

## NAME

autobox::Array::Transform - Autobox methods to transform Arrays

## DESCRIPTION

Autobox high level map-style methods you can call on arrays and
arrayrefs, e.g. map_by(), grep_by(), group_by()
